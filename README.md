# API de Gestión de Grupos
Esta es una aplicación de Express para la gestión de grupos en una base de datos. Proporciona rutas para obtener y crear grupos.
## Rutas
- `GET /`: Obtiene una lista de todos los grupos.
- `POST /`: Crea un nuevo grupo.
## Controladores
Los controladores para estas rutas se encuentran en el archivo `grupos.controller.js`. Asegúrate de implementar la lógica necesaria para manejar las solicitudes y respuestas adecuadamente.
## Uso
1. Clona el repositorio.
2. Instala las dependencias con `npm install`.  
3. Inicia el servidor con `npm start`.
4. Inicia el servidor con `npm run dev` para desarrollo con nodemon.
5. Accede a las rutas mencionadas para gestionar los grupos.
